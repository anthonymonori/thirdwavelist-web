## Contribute

Feel free to take a look at the [open Issues](https://github.com/thirdwavelist/thirdwavelist-web/issues), especially the ones with `good+first+issue` [label](https://github.com/thirdwavelist/thirdwavelist-web/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22) and get going. You will need to get one of the owners to review and accept the PRs before it ends up being released. All help is welcome üôè!

## Build

```sh
npm ci
npm run build
```

## Test locally

```sh
npm run start
```

Please ensure that you don't leave any compiler warnings before submitting a PR.

```sh
npm run test
```
